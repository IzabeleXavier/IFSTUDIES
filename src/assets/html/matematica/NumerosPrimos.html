Numéros Primos

O que são números primos?

Os números primos despertam a curiosidade dos matemáticos e cientistas há mais de mil anos! Desde a antiguidade, busca-se compreender os mistérios por trás dessa sequência numérica considerada infinita, bem como a fórmula ideal para descobrir uma maneira eficaz para encontrá-los em sequências numéricas com muitos dígitos. Para você ficar por dentro do assunto, explicamos o que são esses números e algumas opções usadas para encontrá-los. Confira!

O que é um número primo?

Os números primos são os números naturais que podem ser divididos por apenas dois fatores: o número um e ele mesmo. Vamos conferir alguns exemplos:

  • O número 5 tem apenas dois divisores: o número um e ele mesmo. Portanto, ele é um número primo.
  • O número 5 tem apenas dois divisores: o número um e ele mesmo. Portanto, ele é um número primo.
  • O número 47 tem apenas dois divisores: o número um e ele mesmo. Como resultado, ele é considerado um número primo.
  • O número 80 tem oito divisores: os números um, dois, quatro, cinco, oito, dez, vinte, quarenta e ele mesmo. Logo, ele é um número composto.

Simples, não é mesmo? É importante perceber que o número um não é considerado um número primo, porque ele é divisível apenas por ele mesmo.

Por outro lado, o número dois é o único número primo que também é um número par. Portanto, todos os demais números primos dessa sequência numérica considerada infinita serão obrigatoriamente ímpares. Ainda, com exceção dos números 2 e 5, todos os números primos irão terminar nos algarismos 1, 3, 7 ou 9.

Como relembrar quais são os números primos?

É fundamental não se esquecer quais são os quatro primeiros números primos: dois, três, cinco e sete. Mas não se preocupe! Você não precisa decorar todos os outros. É possível utilizar um algoritmo e, rapidamente, identificar quais são eles, como dentre os cinquenta primeiros números. Acompanhe.

Nós sabemos que o menor número primo é o número dois, mas é importante lembrar que ele é a única exceção entre os números primos. Portanto, pode-se riscar todos os números pares da tabela.

O próximo número primo é o três, o que significa que devemos excluir todos os seus múltiplos; afinal, se um número é múltiplo de três, ele é um número composto, e não primo.

Chegamos ao número cinco - que também é primo. O raciocínio anterior segue correto: deve-se excluir todos os seus múltiplos. Por fim, deve-se fazer o mesmo com o número sete.

Pronto, todos os números que sobraram na sua lista são números primos! Esse sistema para encontrar os números primos chama-se Crivo de Eratóstenes, e foi desenvolvido pelo matemático grego de mesmo nome muitos anos atrás. Esse método pode envolver uma tabela contendo quantos números naturais você desejar. A ideia é sempre encontrar um número primo, e depois excluir todos os seus múltiplos.

Ao final, você terá uma tabela com apenas os números primos do intervalo que você escolheu. E não se esqueça: conhecer quais são esses primeiros números poderá ser muito útil na hora de descobrir se números maiores também são divisíveis por dois únicos fatores, OK?

Como descobrir se um número é primo?

Curiosamente, para descobrir se um número é ou não primo, é necessário dividi-lo pelos próprios números primos. O objetivo é encontrar uma divisão com resto zero - o que significa que o número que está sendo dividido não é primo -, ou uma divisão cujo resultado é menor que o divisor, e com o resto diferente de zero. Nesse caso, ele será considerado um número primo. Acompanhe um exemplo.

O número 139 é um número primo?

  • O número 139 não é par, o que significa que ele não é divisível por dois.
  • Ele também não é divisível pelo número três, porque 1+3+9 é 13, um valor que não é múltiplo de três.
  • Assim como o exemplo anterior, o número não termina em cinco ou em zero; portanto, não é divisível por cinco.
  • Ao realizar a divisão com o número sete, sabemos que o resultado é 19 e o resto é 6. Portanto, o quociente ainda é maior que o divisor.
  • Com o número 11, acontece o mesmo fenômeno. O resultado da divisão é 12 e o resto é 7. O quociente ainda é maior que o divisor.
  • Já com o número 13, as coisas ficam um pouquinho diferentes. Quando dividimos 139, o quociente é 1 e o resto é 9. Aqui, temos um quociente (1) que é menor que o divisor (13), e um resto que é diferente de zero (9).

Assim, podemos afirmar que o número 139 é um primo.

A história dos Números Primos

Os números primos são considerados o maior e mais antigo grupo de números dentro das ciências matemáticas. Euclides foi o matemático responsável por provar a infinidade deste grupo numérico; Erastótenes, por outro lado, foi o primeiro a conseguir encontrar um método simples para listar quais são os números primos dentro de um intervalo - conforme já vimos anteriormente.

A palavra primo refere-se ao termo “primeiro”. A denominação se originou da maneira como a escola pitagórica organizava os números que os matemáticos estudavam, no século 5 a.C. Ao todo, os teóricos criaram três classes de números naturais. São elas:

  • A monad referia-se ao número um, ou a uma única unidade;
  • Os protói aritmói (números primos) ou asynthetói aritmói (números incompostos), referiam-se àqueles que não podem ser gerados pela multiplicação de outros números, como 2, 5, 17;
  • E os deuterói arithmói (números secundários) ou synthetói aritmói (números compostos), diziam respeito aos números que podem ser gerados pela multiplicação de outros números, como 4, 9, 16.

Você sabia?

Os números primos podem ser usados para criptografar informações e transmiti-las de maneira segura através de um canal de comunicação. Um dos métodos utilizados chama-se RSA e é utilizado para manter todo o sistema financeiro seguro no mundo virtual.

Dessa forma, todas as transações on-line que você faz são baseadas na ideia de fatorar números muito grandes em números primos menores, deixando as informações muito mais difíceis de serem decodificadas sem as instruções corretas.
